# Admin Panel

A modern and responsive Admin Dashboard built using TypeScript, SCSS, and Vite. Gives visual representation of the sales, products sold frequently and complete anyalsis.

## 🚀 Features

* **Fast Development:** Powered by Vite for lightning-fast builds and Hot Module Replacement (HMR).
* **Strong Typing:** Written in TypeScript to ensure code reliability and fewer runtime bugs.
* **Modular Styling:** Styled using SCSS for maintainable and organized layout components.
* **Clean UI:** Designed with a focus on data presentation and administrative workflows.

## 🛠️ Tech Stack

* **Build Tool:** Vite
* **Language:** TypeScript
* **Styles:** SCSS / SASS
